# 量化价值深度分析 Skill

## 描述
基于巴菲特价值投资理念的上市公司深度量化分析技能。以"基本面量化"为核心方法论，通过四步递进框架（量化体检→商业模式→管理验证→估值研判），对个股进行系统性扫描、疑点标记、定性验证与保守估值，最终输出结构化投资判断。默认按A股语境组织数据；分析美股等其他市场时，使用当地上市公司披露口径替代A股特有数据。

## 适用场景
- 用户需要对某只A股、美股或其他上市公司股票进行深度价值分析
- 用户要求对个股做"体检""防雷""估值""护城河"等维度的研判
- 用户引用本 Skill 名称或关键词（量化价值分析、基本面量化、深度体检等）

## 触发条件
用户明确要求对某只股票进行深度/全面/量化/价值分析时触发。轻量级的股价查询、单指标比较不触发本 Skill。

## Language Policy

- 默认使用用户提问的语言输出。
- If the user asks in English, produce the analysis report in English.
- 如果用户指定输出语言，以用户指定语言为准。
- 保留 F-Score、B-Score、DCF、moat、margin of safety 等常用投资术语；必要时中英并列。

## 执行流程

### 前置准备
1. 确认分析标的（股票代码、名称和交易市场），若用户未指定，须先确认。
2. 获取基础财务数据（ROE、毛利率、净利率、负债率、PE、PB、股息率、经营现金流、资本支出、总资产、负债、营收、毛利润等），作为量化体检的原始输入。
3. 获取市场与技术参考数据（当前股价、近一年贝塔系数、历史估值分位、行业估值均值等），辅助估值水温和防御属性判断。
4. 联网搜索标的公司最新年报/季报核心数据、行业动态、管理层变动等时效性信息。若运行环境不能联网，必须明确说明数据缺口，并要求用户提供最新公开数据。

> **重要**：所有财务数据必须基于最新可获得的真实公开数据，不得使用过时或估算数据。数据来源不一致时，优先采用公司年报/官方披露。

### 行业与市场适配

- **A股一般工商企业**：可直接使用F-Score、B-Score和两阶段FCF DCF作为主框架。
- **美股或其他市场股票**：用当地披露口径替代A股特有数据；必须标注财报币种、会计准则、股本口径和数据日期。
- **金融、保险、银行、投资控股公司**：F-Score和B-Score仅作疑点扫描，不可机械套用。重点改看股东权益/每股账面价值增长、承保利润、投资收益质量、资本充足性、浮存金成本、回购纪律和长期每股内在价值增长。
- **高现金/高投资组合公司**：DCF估值必须说明是否把超额现金、短期国债、股票投资组合和净债务分开处理；若普通FCF DCF不能解释主要价值来源，必须补充净资产/分部估值/每股账面价值校验。

### 第一部分：量化体检与疑点标记

> 核心目的不是给出结论，而是通过量化指标扫描，**标记出后续定性研究需要重点核查的疑点和线索**。

#### 1.1 估值水温

**操作步骤：**
- 计算当前市盈率（TTM）、市净率（PB）、股息率
- 对比近10年历史数据，明确各指标处于什么分位（用百分比表示）
- 将当前估值水位与行业均值对比

**疑点触发规则：**
- 若估值处于历史极低分位（<10%），标记：
  > 【疑点A：市场为何如此悲观？是短期困境还是长期衰退？后续定性研究必须回答。】
- 若估值处于历史极高分位（>90%），标记：
  > 【疑点A'：市场为何如此乐观？是否透支未来？需核查增长假设的可靠性。】

#### 1.2 财务防雷扫描（F-Score，9分制）

逐项打分，给出明细：

| 类别 | 序号 | 指标 | 判断标准 | 得分 |
|------|------|------|----------|------|
| 盈利能力 | ① | 净利润>0 | 最新年度净利润为正 | 0/1 |
| 盈利能力 | ② | 经营现金流>0 | 最新年度经营现金流为正 | 0/1 |
| 盈利能力 | ③ | 经营现金流>净利润 | 现金流覆盖净利润 | 0/1 |
| 盈利能力 | ④ | ROA同比提高 | 最新年度ROA > 上年度ROA | 0/1 |
| 财务健康 | ⑤ | 长期负债率同比下降 | 最新年度长期负债/总资产 < 上年度 | 0/1 |
| 财务健康 | ⑥ | 流动比率同比提高 | 最新年度流动比率 > 上年度 | 0/1 |
| 财务健康 | ⑦ | 近一年未增发新股 | 近12个月无增发记录 | 0/1 |
| 经营效率 | ⑧ | 毛利率同比提高 | 最新年度毛利率 > 上年度毛利率 | 0/1 |
| 经营效率 | ⑨ | 资产周转率同比提高 | 最新年度总资产周转率 > 上年度 | 0/1 |

**疑点标记规则：**
- ③不通过 → 【疑点B：利润含金量存疑，是正常波动还是盈余质量恶化？】
- ⑧不通过 → 【疑点C：毛利率下滑，是竞争加剧、成本上升，还是产品结构变化？】
- ⑨不通过 → 【疑点D：资产扩张效率下降，新增资产能否产生预期回报？】

**F-Score异常归因：**
若总分≤5分，必须区分：
- **被动失分**：财务结构特殊导致（如重资产行业天然低周转、周期行业ROA波动）
- **真实恶化项**：确实反映经营质量下滑的指标
仅保留真实恶化项作为后续研究线索。

#### 1.3 商业质量速览（量化B-Score，8项，满分10分）

基于公开财务数据计算，评分标准详见 `references/b-score-rubric.md`：

| 维度 | 子项 | 满分 |
|------|------|------|
| 盈利能力真实性 | 净利润稳定性 + 现金流质量 + 资本支出效率 | 3 |
| 商业模式特征 | 毛利率水平 + ROE水平 | 3 |
| 资本配置与治理 | 财务保守度 + 股东回报力度 + 资本纪律 | 4 |

**疑点标记规则：**
- 股东回报<1分 → 【疑点E：管理层是否漠视股东利益？后续需核查管理层资本配置理念。】
- 资本纪律<1分 → 【疑点F：公司是否有过度融资或高溢价并购的历史？后续需核查并购整合效果。】

#### 1.4 防御属性

- 计算近一年贝塔（β）系数
- 整理近年分红比例及股息支付率变化趋势
- 判断标的的防御/进攻属性

### 第二部分：商业模式与护城河

> 以下每一项分析，必须结合第一部分标记的疑点，用定性证据进行验证或推翻。

#### 2.1 价值创造与收入结构

**操作步骤：**
1. 按业务板块和渠道，拆分最新的收入与利润构成（引用年报数据）
2. 计算各板块的收入占比、利润占比、毛利率
3. 分析各板块的增长趋势与贡献度

**疑点关联：**
- 若【疑点C】被标记 → 重点分析各业务板块毛利率变化趋势，找出下滑的具体环节（原料成本？制造费用？产品结构？），初步判断结构性 vs 周期性

#### 2.2 客户与价值主张

**必须回答：**
1. 核心用户群体是谁？解决了什么根本性需求或痛点？
2. 在竞争对手中，客户为何选择它？价值主张关键是品牌、成本、技术、渠道还是综合服务？
3. 客户转换成本有多高？是否存在锁定效应？

**疑点关联：**
- 若【疑点A】被标记 → 追问：价值主张是否在弱化？客户粘性是否在下降？市场份额是否被侵蚀？

#### 2.3 护城河评估

**评估框架：**

| 护城河来源 | 强度定义 | 判断依据 |
|------------|----------|----------|
| 品牌溢价 | 客户愿为品牌支付持续溢价，且溢价长期稳定 | 定价权+品牌认知度数据 |
| 专利/技术壁垒 | 核心技术不可复制，且持续投入维持领先 | 研发投入占比+专利数量+技术迭代速度 |
| 成本优势 | 单位成本持续低于同行，且优势可维持 | 成本结构对比+规模效应数据 |
| 网络效应 | 用户增加直接提升产品价值 | 用户数增长+单用户价值数据 |
| 特许经营权 | 政策/牌照形成的合法垄断 | 牌照稀缺性+政策稳定性 |

**综合评级：** 极宽 / 宽 / 窄 / 无

**疑点关联：**
- 若【疑点B】或【疑点C】被标记 → 评估：护城河是否在变窄？竞争优势是否在侵蚀利润？
- 若护城河为"窄"或"无" → 必须明确说明如何在缺乏护城河的情况下维持当前盈利能力

### 第三部分：历史、管理层与资本配置

> 以下每一项分析，必须结合第一部分标记的疑点，验证管理层的能力和诚信。

#### 3.1 企业发展史与战略转折点

**必须梳理：**
1. 历史性机遇：行业改革、上市、重大并购、国家战略支持等
2. 重大危机：行业巨震、经营危机、安全事故等，管理层的应对及效果
3. 战略转折点的决策质量评估

**疑点关联：**
- 若【疑点F】被标记 → 重点核查历史上重大并购的整合效果、股权融资的使用效率

#### 3.2 管理层能力与诚信

**核查维度：**
1. 核心领导层的管理哲学和战略眼光
2. 企业文化核心：是否有"利润之上"的追求？有无真实案例？
3. 管理层薪酬结构是否与股东利益一致

**疑点关联：**
- 若【疑点E】被标记 → 追问：管理层是否在年报或公开讲话中表达过对股东回报的态度？实际行为（分红记录、回购行为、薪酬结构）是否与表态一致？

#### 3.3 逆境中的管理层行为验证

**核心验证逻辑：**
在行业下行或公司业绩困境时，管理层选择的是：
- 缩减研发、品牌等长期投入以维系短期利润？→ 负面信号
- 逆势加码、捍卫长期竞争力？→ 正面信号

**必须引用真实历史案例**，用公司具体行动验证管理层的品质和韧性。

#### 3.4 内部组织变革与效率

**核查内容：**
1. 为适应内外部变化，公司经历了哪些重要的组织变革？
2. 变革如何影响了效率和竞争力？
3. 执行效果的量化证据

**疑点关联：**
- 若【疑点D】被标记 → 核查：是否有组织架构臃肿、人效下降问题？管理层的组织变革是否对症？执行效果如何？

#### 3.5 完整版B-Score评定（15分制）

在量化版（10分）基础上，增加以下定性维度（5分）：

| 维度 | 评分标准 | 满分 |
|------|----------|------|
| 客户粘性与定价权 | 成瘾/社交/刚需且替换成本极高=1.5；品牌忠诚但竞争充分=0.75；同质化竞争=0 | 1.5 |
| 周期穿越能力 | 近一轮行业下行期，营收降幅<行业均值50%=1.5；优于行业但<50%=1；同步=0.5；更差=0 | 1.5 |
| 研发/品牌投入持续度 | 利润承压时逆势增加=1.5；微降=1；大幅削减=0 | 1.5 |
| 管理层诚信与能力 | 基于历史行为验证：无污点且多次正确决策=1；基本可信=0.5；有诚信瑕疵=0 | 1 |

> **此评分必须在所有深度分析完成后进行**，不得跳过分析直接评分。

### 第四部分：估值、风险与综合研判

#### 4.1 内在价值估算

**默认方法：两阶段自由现金流折现模型**

若标的是金融、保险或投资控股公司，必须先判断普通FCF DCF是否适用。若不适用，采用“经营业务DCF + 投资组合/现金净值 + 保险浮存金质量 + 每股账面价值校验”的组合估值，并明确说明普通DCF结果只作为压力测试或下限参考。

**必须明确列出所有假设：**
1. 未来3-5年增速假设（必须保守，基于前三部分分析得出，不得随意取值）
2. 永续增长率假设（通常不超过3%）
3. 折现率（WACC）的推导过程

**必须完整展示：**
- 阶段一（高增长期）每年的自由现金流预测
- 阶段二（永续期）的终值计算
- 折现到现值的完整过程
- 每股内在价值的计算

**安全边际计算：**
- 安全边际 = (内在价值 - 当前股价) / 内在价值 × 100%

#### 4.2 核心风险与正面驱动

**必须给出：**
- 未来3-5年盈利最关键的1个正面驱动因素
- 未来3-5年盈利最关键的1个负面风险因素
- Top 3最具体、可能影响投资判断的风险

- 13-15分：⭐⭐⭐⭐⭐（卓越）
- 10-12分：⭐⭐⭐（优秀）
- 7-9分：⭐⭐（中等）
- 4-6分：⭐（较弱）
- 0-3分：❌（危险）

**综合建议判断标准：**
- 安全边际>30% 且 B-Score≥10 且 无致命疑点 → 值得关注
- 安全边际15-30% 且 B-Score≥7 → 需等待更低价位
- 安全边际<15% 或 B-Score<7 → 不值得当前价位介入

## 数据获取指引

1. **基础财务数据**：优先采用公司年报、季报、交易所公告、巨潮资讯、公司投资者关系材料等官方披露；其次采用可信金融数据源。
2. **市场数据**：获取当前股价、PE(TTM)、PB、股息率、历史估值分位和贝塔系数；无法获取实时数据时，标注数据日期。
3. **时效性信息**：必须联网核查公司年报/季报、行业动态、管理层变动、监管处罚、重大并购、政策变化等。
4. **行业对比**：选择3-5家业务可比公司，比较估值、毛利率、ROE、资产周转率和周期表现。
5. **历史估值分位**：若没有可直接引用的10年分位数据，可用可获得的历史PE/PB序列计算；数据不完整时说明样本周期。

## 注意事项

1. **保守原则**：所有预测必须保守，所有计算必须严谨，判断必须基于真实事实
2. **追问原则**：对每一个商业现象必须追问"为什么"和"这意味着什么"
3. **诚实原则**：数据不足以完美回答时，基于最可靠信息给出尽可能完整且诚实的分析，明确标注不确定性
4. **疑点闭环**：第一部分标记的每个疑点，必须在后续部分得到验证或推翻，不可遗漏
5. **禁止跳步**：不得跳过分析直接给出评分或结论
6. **引用来源**：关键数据和结论必须标注来源
7. **计算严谨**：复杂数学运算（乘除、折现等）必须通过代码完成，不得心算
