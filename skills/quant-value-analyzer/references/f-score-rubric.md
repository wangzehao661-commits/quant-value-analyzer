# F-Score 财务防雷评分细则

## 评分总览

F-Score 采用 Piotroski 9分制，逐项判断通过/不通过，满分9分。

---

## 逐项评分标准

### 一、盈利能力（4项）

#### ① 净利润 > 0
- **判断**：最新年度归母净利润是否为正值
- **数据来源**：利润表 → 归属于母公司所有者的净利润
- **通过**：净利润 > 0 → 得1分
- **不通过**：净利润 ≤ 0 → 得0分

#### ② 经营现金流 > 0
- **判断**：最新年度经营活动产生的现金流量净额是否为正值
- **数据来源**：现金流量表 → 经营活动产生的现金流量净额
- **通过**：经营现金流 > 0 → 得1分
- **不通过**：经营现金流 ≤ 0 → 得0分

#### ③ 经营现金流 > 净利润
- **判断**：经营现金流是否覆盖净利润（衡量利润含金量）
- **数据来源**：现金流量表 + 利润表
- **通过**：经营现金流 > 净利润 → 得1分
- **不通过**：经营现金流 ≤ 净利润 → 得0分
- **疑点触发**：不通过时标记【疑点B】

> **特殊情况处理**：
> - 资源类企业（石油、煤炭等）因资本支出大，经营现金流可能天然高于净利润，此指标通常通过
> - 重资产折旧企业同理
> - 若此项不通过，需区分：是应收账款暴增、存货积压，还是正常的资本支出周期

#### ④ ROA同比提高
- **判断**：最新年度总资产收益率是否高于上年度
- **计算**：ROA = 净利润 / 总资产（期末或平均）
- **通过**：最新ROA > 上年ROA → 得1分
- **不通过**：最新ROA ≤ 上年ROA → 得0分

---

### 二、财务健康（3项）

#### ⑤ 长期负债率同比下降
- **判断**：最新年度长期负债/总资产 是否低于上年度
- **计算**：长期负债率 = 非流动负债合计 / 资产总计
- **通过**：最新长期负债率 < 上年长期负债率 → 得1分
- **不通过**：最新长期负债率 ≥ 上年长期负债率 → 得0分

> **行业调整**：金融、地产等高杠杆行业此指标天然偏低，需结合行业特征判断

#### ⑥ 流动比率同比提高
- **判断**：最新年度流动比率是否高于上年度
- **计算**：流动比率 = 流动资产 / 流动负债
- **通过**：最新流动比率 > 上年流动比率 → 得1分
- **不通过**：最新流动比率 ≤ 上年流动比率 → 得0分

#### ⑦ 近一年未增发新股
- **判断**：近12个月内是否有增发新股行为
- **数据来源**：公司公告、股本变动
- **通过**：无增发 → 得1分
- **不通过**：有增发 → 得0分

> **注意**：
> - 仅指增发融资（定向增发、公开增发），不含可转债转股、股权激励行权等
> - 增发不一定负面，若增发用于高回报项目且价格合理，可在分析中备注

---

### 三、经营效率（2项）

#### ⑧ 毛利率同比提高
- **判断**：最新年度毛利率是否高于上年度
- **计算**：毛利率 = (营业收入 - 营业成本) / 营业收入
- **通过**：最新毛利率 > 上年毛利率 → 得1分
- **不通过**：最新毛利率 ≤ 上年毛利率 → 得0分
- **疑点触发**：不通过时标记【疑点C】

> **分析要点**：毛利率下降可能是：
> 1. **成本推动**：原材料价格上涨（如石油公司的原油成本）
> 2. **竞争加剧**：被迫降价
> 3. **产品结构变化**：低毛利业务占比提升
> 4. **会计调整**：成本核算方法变更

#### ⑨ 资产周转率同比提高
- **判断**：最新年度总资产周转率是否高于上年度
- **计算**：总资产周转率 = 营业收入 / 总资产（期末或平均）
- **通过**：最新周转率 > 上年周转率 → 得1分
- **不通过**：最新周转率 ≤ 上年周转率 → 得0分
- **疑点触发**：不通过时标记【疑点D】

> **行业特性**：
> - 重资产行业（石油、电力、基建等）天然低周转，此指标失分常见
> - 需区分：是新增资产尚未产生回报（扩张期正常），还是资产运营效率真实下降

---

## F-Score 总分解读

| 分数 | 含义 | 后续动作 |
|------|------|----------|
| 8-9 | 财务状况强劲 | 无重大疑点，可重点看估值 |
| 6-7 | 财务状况良好 | 个别指标需关注，结合定性分析 |
| 4-5 | 财务状况一般 | 必须仔细归因，区分被动失分与真实恶化 |
| 0-3 | 财务状况堪忧 | 除非有明确的周期反转逻辑，否则建议回避 |

## F-Score 异常归因模板

当 F-Score ≤ 5 时，必须填写：

| 失分项 | 被动失分（行业/结构原因） | 真实恶化（经营质量下降） | 需后续研究 |
|--------|--------------------------|--------------------------|------------|
| ④ ROA下降 | □ 行业周期下行 | □ 竞争力减弱 | □ |
| ⑤ 负债率上升 | □ 正常扩张融资 | □ 偿债压力增大 | □ |
| ⑥ 流动比率下降 | □ 季节性波动 | □ 流动性风险 | □ |
| ⑧ 毛利率下降 | □ 成本周期性上行 | □ 竞争/结构恶化 | □ |
| ⑨ 周转率下降 | □ 新项目投产初期 | □ 运营效率下降 | □ |

注：仅勾选一项，如两者皆有可能，在"需后续研究"中标注。
