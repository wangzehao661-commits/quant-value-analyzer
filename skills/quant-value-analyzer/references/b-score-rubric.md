# B-Score 评分细则（量化版 + 完整版）

## 一、量化速览版（可自动计算，8项，满分10分）

### 1. 盈利能力真实性（满分3分）

#### 1.1 净利润稳定性（1分）
| 条件 | 得分 |
|------|------|
| 近5年净利润均为正，且标准差/均值 < 30% | 1 |
| 近5年净利润均为正，但波动较大（30% ≤ 标准差/均值 < 60%） | 0.5 |
| 近5年有亏损年度，或波动极大（标准差/均值 ≥ 60%） | 0 |

#### 1.2 现金流质量（1分）
| 条件 | 得分 |
|------|------|
| 近3年经营现金流/净利润 均 > 1.0 | 1 |
| 近3年经营现金流/净利润 均值在 0.7-1.0 之间 | 0.5 |
| 近3年经营现金流/净利润 均值 < 0.7 | 0 |

#### 1.3 资本支出效率（1分）
| 条件 | 得分 |
|------|------|
| 近3年（经营现金流 - 资本支出）/经营现金流 均 > 50% | 1 |
| 近3年均值在 30%-50% 之间 | 0.5 |
| 近3年均值 < 30%（资本支出吞噬大部分现金流） | 0 |

---

### 2. 商业模式特征（满分3分）

#### 2.1 毛利率水平（1.5分）
| 条件 | 得分 |
|------|------|
| 最新年度毛利率 > 40% | 1.5 |
| 最新年度毛利率在 20%-40% 之间 | 0.75 |
| 最新年度毛利率 < 20% | 0 |

#### 2.2 ROE水平（1.5分）
| 条件 | 得分 |
|------|------|
| 近3年平均ROE > 15% | 1.5 |
| 近3年平均ROE在 8%-15% 之间 | 0.75 |
| 近3年平均ROE < 8% | 0 |

---

### 3. 资本配置与治理（满分4分）

#### 3.1 财务保守度（1分）
| 条件 | 得分 |
|------|------|
| 资产负债率 < 40% | 1 |
| 资产负债率在 40%-60% 之间 | 0.5 |
| 资产负债率 > 60% | 0 |

> 注：金融类企业不适用此标准，需单独调整。

#### 3.2 股东回报力度（1.5分）
| 条件 | 得分 |
|------|------|
| 近3年平均股息率 > 3%，且有分红增长趋势 | 1.5 |
| 近3年平均股息率在 1.5%-3% 之间，分红较稳定 | 0.75 |
| 近3年平均股息率 < 1.5%，或分红不稳定/中断 | 0 |

#### 3.3 资本纪律（1.5分）
| 条件 | 得分 |
|------|------|
| 近5年无高溢价并购（溢价率>50%），无频繁增发，融资用途与承诺基本一致 | 1.5 |
| 有少量并购或融资行为，但整体审慎，溢价合理 | 0.75 |
| 存在高溢价并购、频繁增发、或融资用途与承诺严重不符 | 0 |

---

## 二、完整版（需定性判断，15分制）

在量化版（10分）基础上，增加以下4个定性维度（满分5分）：

### 4. 客户粘性与定价权（1.5分）
| 条件 | 得分 |
|------|------|
| 产品/服务具有成瘾性/社交属性/刚需且替换成本极高，客户流失率极低 | 1.5 |
| 有品牌忠诚度但竞争充分，定价权有限 | 0.75 |
| 同质化竞争，客户无粘性，价格敏感 | 0 |

**判断依据：**
- 成瘾性：如烟草、游戏
- 社交属性：如即时通讯、支付网络
- 刚需+替换成本：如企业ERP、操作系统
- 品牌忠诚：如消费品品牌，但存在替代品
- 同质化：如大宗商品、基础制造

### 5. 周期穿越能力（1.5分）
| 条件 | 得分 |
|------|------|
| 近一轮完整行业下行期，公司营收降幅 < 行业均值降幅的50% | 1.5 |
| 公司营收降幅优于行业，但 < 50% 优势 | 1 |
| 与行业基本同步 | 0.5 |
| 营收降幅超过行业均值 | 0 |

**操作指引：**
- 选取最近一轮行业下行周期（如2014-2016石油低谷、2018年地产调整等）
- 对比同行业3-5家主要竞争对手的营收变化
- 如行业数据不易获取，可用行业GDP增速变化替代

### 6. 研发/品牌投入持续度（1.5分）
| 条件 | 得分 |
|------|------|
| 利润承压时期（营收或利润同比下滑年份），研发费用率或品牌投入反而提升 | 1.5 |
| 利润承压时期，研发/品牌投入微降（降幅<10%） | 1 |
| 利润承压时期，研发/品牌投入大幅削减（降幅≥10%） | 0 |

**操作指引：**
- 选取公司近5年中利润最差的1-2年
- 对比该年度与正常年度的研发费用率（研发费用/营收）
- 品牌投入可用销售费用率近似替代（如适用）

### 7. 管理层诚信与能力（1分）
| 条件 | 得分 |
|------|------|
| 无诚信污点，且在关键战略节点做出正确决策（有历史案例支撑） | 1 |
| 基本可信，无重大诚信问题，决策质量尚可 | 0.5 |
| 存在诚信瑕疵（如财务造假嫌疑、关联交易问题、承诺不兑现等） | 0 |

**判断依据：**
- 是否有监管处罚记录
- 是否有财务重述或审计意见异常
- 承诺与兑现的对比（如业绩承诺、回购承诺）
- 关联交易是否公允

## 三、B-Score星级对应

| 分数区间 | 星级 | 含义 |
|----------|------|------|
| 13-15 | ⭐⭐⭐⭐⭐ | 卓越：极少见的优质标的 |
| 10-12 | ⭐⭐⭐ | 优秀：具备显著竞争优势 |
| 7-9 | ⭐⭐ | 中等：有亮点但也有明显短板 |
| 4-6 | ⭐ | 较弱：多数维度不达标 |
| 0-3 | ❌ | 危险：基本面存在严重问题 |

---

## 四、评分注意事项

1. **逐项评分，给出依据**：每个子项的得分必须有对应的事实或数据支撑，不得凭感觉打分。
2. **行业调整**：部分指标需考虑行业特性（如金融企业资产负债率、资源型企业毛利率天然偏低），在评分时注明调整理由。
3. **趋势优先**：同一指标处于临界值时，趋势向好加分，趋势恶化减分。
4. **保守原则**：信息不足时，取较低分数，不做乐观假设。
5. **完整版必须在定性分析后评分**：客户粘性、周期穿越、研发持续度、管理层诚信这4项，必须在完成对应的定性研究后才能打分，严禁跳步。
