# DCF 估值操作指引

## 模型选择：两阶段自由现金流折现

### 为什么用两阶段
- 第一阶段（3-5年）：反映公司近中期的可预测增长
- 第二阶段（永续期）：反映长期稳定增长假设
- 比单阶段更灵活，比三阶段更保守

### 适用性检查

先判断普通FCF DCF是否能解释公司的主要价值来源。

- 一般工商企业：通常适用。
- 金融、保险、银行、投资控股公司：通常不宜单独使用普通FCF DCF。应补充净资产、投资组合市值、浮存金成本、分部经营利润和每股账面价值增长校验。
- 高现金/高投资组合公司：DCF中必须单独处理超额现金、短期国债、股票投资组合和净债务，避免把金融资产产生的价值遗漏或重复计算。
- FCF为负或极不稳定公司：不要强行DCF；改用正常化盈利、分部估值、P/B或重置成本等方法。

---

## 第一步：确定自由现金流（FCF）

### 公式
```
FCF = 经营活动现金流净额 - 资本支出
```

### 数据获取
- 经营活动现金流净额：现金流量表
- 资本支出：现金流量表 → 购建固定资产、无形资产和其他长期资产支付的现金

### 处理要点
1. 取近3-5年FCF均值作为基准，避免单年异常值
2. 若近3年FCF趋势明显（持续增长/下降），可加权处理（近1年权重>前2年）
3. FCF为负的企业，本模型不适用，需改用其他估值方法

---

## 第二步：设定增长率假设

### 阶段一（高增长期）增长率 g1

**必须基于前三部分分析得出，不得随意取值。推导逻辑：**

1. **历史增速参考**：近5年营收/净利润CAGR
2. **行业增速参考**：行业未来3-5年预期增速（搜索获取）
3. **公司份额趋势**：市占率提升/维持/下降
4. **产能扩张**：在建工程/计划投资是否支撑增长

**保守调整规则：**
- 若历史增速 > 15%，取 min(历史增速 × 0.7, 15%) 作为g1
- 若历史增速 5%-15%，取历史增速 × 0.8 作为g1
- 若历史增速 < 5%，直接取历史增速作为g1
- 任何情况下，g1 不超过 20%

### 阶段二（永续期）增长率 g2

- 通常取 2%-3%，不超过GDP长期增速
- 中国市场建议取 2.5%
- 若行业已进入成熟期，取 2%
- 若行业仍在成长，取 3%

---

## 第三步：确定折现率（WACC）

### 公式
```
WACC = E/(E+D) × Re + D/(E+D) × Rd × (1-T)
```

### 各参数取值

| 参数 | 取值方法 | 默认值 |
|------|----------|--------|
| Re（股权成本） | CAPM: Rf + β × (Rm - Rf) | — |
| Rf（无风险利率） | 中国10年期国债收益率 | 搜索获取 |
| β（贝塔系数） | 从 stock-analysis 技能获取 | — |
| Rm - Rf（市场风险溢价） | A股历史数据 | 6%-7% |
| Rd（债务成本） | 公司有息负债利率 | 4%-5% |
| T（税率） | 公司实际所得税率 | 年报数据 |
| E/(E+D) | 股权市值占比 | 计算得出 |
| D/(E+D) | 债务市值占比 | 计算得出 |

### 简化处理
若数据不足以精确计算WACC，可使用行业经验值：
- 稳定行业（公用事业、银行）：8%-9%
- 一般行业（制造、消费）：9%-11%
- 高波动行业（科技、周期）：11%-13%

---

## 第四步：计算过程

### 阶段一：逐年折现
```
FCF_1 = FCF_0 × (1 + g1)
FCF_2 = FCF_1 × (1 + g1)
...
FCF_n = FCF_{n-1} × (1 + g1)

PV(FCF_1) = FCF_1 / (1 + WACC)^1
PV(FCF_2) = FCF_2 / (1 + WACC)^2
...
PV(FCF_n) = FCF_n / (1 + WACC)^n
```

### 阶段二：终值折现
```
TV = FCF_n × (1 + g2) / (WACC - g2)
PV(TV) = TV / (1 + WACC)^n
```

### 企业价值
```
EV = Σ PV(FCF_i) + PV(TV)
```

### 每股内在价值
```
每股内在价值 = (EV - 净债务) / 总股本
净债务 = 总有息负债 - 现金及等价物
```

### 安全边际
```
安全边际 = (每股内在价值 - 当前股价) / 每股内在价值 × 100%
```

## 注意事项

1. **所有计算必须通过代码完成**，不得心算或粗略估算
2. **增长率假设必须列出推导过程**，不接受"假设增长X%"而无依据
3. **WACC各参数必须标注来源**
4. **FCF为负的企业不适用此模型**，应改用：
   - 重资产企业：P/B + 重置成本法
   - 周期企业：正常化盈利法
   - 亏损企业：暂不估值，仅做定性分析
5. **最终估值结果需给出合理性校验**：
   - 对比当前PE/PB隐含的增速是否一致
   - 对比同行业可比公司估值水平
